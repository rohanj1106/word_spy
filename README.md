# Word Spy — Word Puzzle Game

A word puzzle game built with Flutter, inspired by Word Trip.

## Tech Stack

- **Frontend:** Flutter + Flame
- **Backend:** Supabase (auth, database, edge functions)
- **AI:** Claude / OpenAI API for puzzle generation

## Getting Started

```bash
flutter pub get
flutter run
```
